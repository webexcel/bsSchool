### Clone the repo
```> git clone https://github.com/webexcel/st_parentapp.git```

### install packages
```> npm install --force```

### To run locally
```> ionic serve```

### To run the application in android studio
```> npx cap add android```

```> ionic build```

```> npx cap sync android```

```> npx cap copy```

```> npx cap open android```
